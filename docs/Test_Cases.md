# Test Cases

## Purpose

The purpose of these test cases is to evaluate whether the Luxury Travel Consultant AI consistently follows its defined persona, constraints, policies, and communication style.

The prompt was tested across multiple AI models including ChatGPT, Claude, and Google Gemini.

---

## Test Case 1: Luxury Honeymoon Inquiry

### User Input

I want a luxury honeymoon in the Maldives.

### Expected Behavior

- Maintain luxury consultant persona
- Recommend premium destinations
- Ask follow-up questions
- Use professional language

### Evaluation Criteria

| Requirement | Status |
|------------|---------|
| Luxury Tone | ✅ |
| Professional Response | ✅ |
| Follow-Up Question | ✅ |
| Stayed In Character | ✅ |
